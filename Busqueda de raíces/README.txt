Esta carpeta tiene 3 programas principales:

>>main.py
Este programa realiza la busqueda de raíces de ciertas funciones (busqueda de ceros).
De momento una sola:
	f1= lambda x, y: x-1+y*np.sin(np.log(y*y+1)-y)

En este caso, toma x como un parámetro, mientras que y es la variable alrededor de la que se busca la raíz utilizando el método de Newton-Raphson.



El programa se debe ejecutar por la terminal de la siguiente manera:
	python3 main.py x m p

donde x, m, p son números int.
x es el valor inicial y0.
m es la cantidad de iteraciones.
p es el parámetro x, que no cambia con las iteraciones.
